# Security Policy

## Educational Scope

The tools, models, and code contained in this repository are developed strictly for academic, research, and educational purposes. Adversarial attack implementations and prompt injection demonstrations are intended exclusively for understanding AI security vulnerabilities in controlled environments.

## Reporting a Vulnerability

If you discover a security vulnerability within this project itself, please do not disclose it publicly.

Instead, open an issue labeled `security` or contact the repository owner directly. Given the educational nature of the project, reasonable effort will be made to remediate architectural or dependency vulnerabilities.

## Responsible Use

By cloning or downloading materials from this repository, you agree to:

1. Not use adversarial attack techniques against ML systems you do not own or have explicit permission to test.
2. Not use prompt injection demonstrations to exploit commercial AI services.
3. Not use the code to generate harmful content, deepfakes, or misleading AI outputs.
4. Assume full responsibility for compliance with all applicable local and international laws.
5. Acknowledge that these implementations are simplified for educational purposes and are not production-grade attack tools.
